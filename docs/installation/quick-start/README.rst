Prerequisites
#############

In order to use the Raiden Wizard and Raiden you'll need:

-  A computer running **Linux** or **macOS**
-  `MetaMask <https://metamask.io>`__ and an account with at least
   ``0.13`` ETH
-  An `Infura <https://infura.io>`__ ID
