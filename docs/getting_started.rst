Getting Started
###############

Now that your Raiden node is up and running, you can choose from the following tutorials, depending on your Raiden use case:

See :doc:`Using Raiden on Mainnet <using-raiden-on-mainnet/overview>` for getting started to use Raiden in the Ethereum mainnet.
The :doc:`Raiden Web Interface <the-raiden-web-interface/the-raiden-web-interface>` is the easiest way to use Raiden interactively. 
For programmatic use of Raiden, see :doc:`The Raiden API Tutorial <raiden-api-1/api-tutorial/README>`.
