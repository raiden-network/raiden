Payment-Channel Network for Ethereum

Raiden is a technology that leverages off-chain state networks to extend Ethereum with some nice properties for asset transfers:

  • Scalable

    it scales linearly with the number of participants (1,000,000+ transfers per second possible)

  • Fast

    Transfers are confirmed and final within the fraction of a second

  • Confidential

    Single transfers don’t show up in the global shared ledger

  • Interoperable

    Works with any token that follows Ethereum’s standardized token API.

  • Low Fees

    Transaction fees can be 7 orders of magnitude lower than on the blockchain

  • Micro-payments

    Low transaction fees allow to efficiently transfer tiny values.
